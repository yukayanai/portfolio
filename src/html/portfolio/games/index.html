Gaming Ecommerce Website

Revamping the user experience of an e-commerce gaming website through a comprehensive redesign project

Header photo by Sora Khan on Unsplash

Background

Role: I undertook the role of a UI/UX designer.
Methods used: We used various methods such as current state analysis by creating a journey map, conducted user interviews, mind mapping, user testing.
Tools used: Google Analytics, Hotjar, Miro, Figma
Team members: PO, UX designer, a frontend developer, a backend developer, a QA developer.
Stakeholders: Customers, Marketing manager.

Internal Application

This was my first project at EB Games. The goal was to redesign and restructure the merchandiser's application, while maintaining its value.

In 7 years of selling and merchandising products online, the business has evolved and grown to the extent of merchandising the same products three times, and triplicating the effort for development and business to build and maintain three applications.

Objectives:

  • One interface
  • Product taxonomy Structure
  • Create consistent product data
  • Group logically
  • Ability to edit products in bulk
  • Remove redundant data and attributes
  • Reduce Dev support

Internal application UI design

Redesign website

The primary goal for this project was to create a responsive website that can be consumed by any devices and to create a unique discovery journey to customers.

UX Research

Customer journey mapping and identify problems with the current EB Games and Zing websites
