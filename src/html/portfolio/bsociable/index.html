Bsociable

At Bsociable, I helped with providing feedback to improve their UI & UX for their app. I worked closely with the devs to bring the app to life focusing on the onboarding journey for the new users.

Wireframes

Proposed wireframes for the onboarding process

UI Design

The Problem

This social event planner app has so many different features that is not clearly visible to users. One of the many reasons was, it did not have an onboarding process for new users to learn about the different features they could use to create an efficient way to plan their events.

The Objectives

∙   Create a simple step by step process to add profile details for new users
∙   Showcase and prioritise the features of Bsociable
∙   Create consistency throughout the app
∙   Group features in a logical manner
∙   Introduce friendly and fun elements to enhance user experience
