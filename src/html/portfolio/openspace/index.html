Open Colleges Design System

Modernise the Open Colleges learning experiences and build and sustain a design system.

Background

OpenSpace is an online campus for students to access all their learning materials including videos and digital resources, assessments, grades and feedback. Through OpenSpace, students can also connect with their Trainers, Assessors, Program Managers, online support network and other students. As a designer in Open Colleges, we set out to build and sustain a design system. The design system would include components and assets needed to create a visual language and foundational component library.

Role: UX Designer
Methods and Tools: Sketch, Adobe Illustrator
Team members: A Product Owner, and 2 Developers.

Design Iteration

With a number of design iterations, we worked together with the Developers to update the styles of OpenSpace component by component.

Design System

I took all our brand assets and developed it into a Sketch component library, so that it was easy for designers across the organisation to have the latest brand resources.

Iconography

Previously, we had icons all over the place in inconsistent styles. In this project, I created a set of pixel-aligned icons that would work for most use cases across the Open Colleges learning platform.
